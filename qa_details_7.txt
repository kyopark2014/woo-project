# 테스트 케이스: 부가세 참고자료 메일 전송 V3 - 배민 부담 가게배달팁 '기타매출'로 포함/제공

## TC-01: 부가세 참고자료 메일 전송 시 배민 부담 가게배달팁이 기타매출에 포함되는지 확인

**선행 조건:**
- 테스트 환경이 구성되어 있음
- 테스트용 거래처번호가 준비되어 있음
- 배민 부담 가게배달팁이 발생한 거래 데이터가 존재함

**테스트 단계:**
1. 부가세 참고자료 메일 전송 V3 API를 호출
2. 요청 파라미터에 거래처번호와 조회 기간(startMonth, endMonth) 설정
3. 메일 전송 요청 후 응답 확인
4. 전송된 메일 내용 확인

**예상 결과:**
- API 응답이 성공적으로 반환됨
- 전송된 메일의 부가세 참고자료에 배민 부담 가게배달팁이 '기타매출' 항목에 포함되어 있음
- 기타매출 금액에 배민 부담 가게배달팁 금액이 정확히 합산되어 있음

**검증 항목:**
- 메일 내 기타매출 항목에 배민 부담 가게배달팁 금액이 포함되어 있는지 확인
- 기타매출의 공급가액, 부가세, 합계 금액이 올바르게 계산되었는지 확인

## TC-02: 배민 부담 가게배달팁이 있는 경우와 없는 경우의 기타매출 비교 검증

**선행 조건:**
- 테스트 환경이 구성되어 있음
- 동일한 거래처에 대해 배민 부담 가게배달팁이 있는 거래와 없는 거래 데이터가 준비되어 있음

**테스트 단계:**
1. 배민 부담 가게배달팁이 없는 기간에 대해 부가세 참고자료 메일 전송 V3 API 호출
2. 배민 부담 가게배달팁이 있는 기간에 대해 부가세 참고자료 메일 전송 V3 API 호출
3. 두 메일의 기타매출 항목 비교

**예상 결과:**
- 배민 부담 가게배달팁이 있는 기간의 메일에서는 기타매출에 해당 금액이 포함됨
- 배민 부담 가게배달팁이 없는 기간의 메일에서는 기타매출에 해당 금액이 포함되지 않음
- 두 기간의 기타매출 차이가 배민 부담 가게배달팁 금액과 일치함

**검증 항목:**
- 두 기간의 기타매출 금액 차이 확인
- 배민 부담 가게배달팁 금액이 정확히 기타매출에 반영되었는지 확인

## TC-03: 부가세 참고자료 메일의 매출 상세 내역에서 배민 부담 가게배달팁 확인

- 결제수단이 '배민 부담 가게배달팁'으로 정확히 표시됨

**검증 항목:**
- 매출 상세 내역에서 배민 부담 가게배달팁 거래가 올바르게 표시되는지 확인
- 매출구분이 '기타매출'로 표시되는지 확인
- 결제수단이 '배민 부담 가게배달팁'으로 표시되는지 확인
- 공급가액, 부가세, 합계 금액이 올바르게 계산되었는지 확인

## TC-04: 부가세 참고자료 메일에서 배민 부담 가게배달팁의 부가세 계산 정확성 확인

**선행 조건:**
- 테스트 환경이 구성되어 있음
- 배민 부담 가게배달팁이 발생한 거래 데이터가 존재함

**테스트 단계:**

4. 전송된 메일에서 배민 부담 가게배달팁의 부가세 계산 확인

- 공급가액과 부가세의 합이 배민 부담 가게배달팁 총액과 일치하는지 확인
